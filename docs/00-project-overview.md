## Design System

### Color Palette
- **Primary Green**: `#2C5F2D` - Growth, Hope, Trust
- **Secondary Orange**: `#FF6B35` - Energy, Action, Change
- **Accent Blue**: `#0077B6` - Credibility, Professional
- **Neutral Dark**: `#2D3142` - Text color
- **Neutral Light**: `#F8F9FA` - Backgrounds
- **Success Green**: `#06D6A0` - Impact metrics, success states

### Typography
- **Headings**: Poppins (Google Fonts) - Bold, 600-700 weight
- **Body Text**: Inter (Google Fonts) - Regular, 400 weight
- **Hindi/Marathi**: Noto Sans Devanagari (Google Fonts)

### Design Principles
- Clean, modern, spacious layouts
- Card-based design with subtle shadows
- Gradient overlays on hero images
- Rounded corners (8px border-radius)
- Ample white space for readability
- Mobile-first responsive design
- Smooth animations and transitions

## Key Features
- Multi-language support (English, Hindi, Marathi)
- Responsive design (mobile, tablet, desktop)
- SEO optimized content
- Accessibility compliant
- Fast loading times
- Social media integration
- Newsletter signup
